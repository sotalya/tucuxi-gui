<p>
The current state of this software allows the user to:
<ul>
        <li>Create fake patients</li>
        <li>Select a medical drug</li>
        <li>Select a dosage</li>
        <li>Add patient's covariates</li>
        <li>Add drug concentration measures</li>
        <li>Propose a drug dosage adjustment</li>
        <li>Save and load an interpretation</li>
        <li>Generate a PDF report (not customized)</li>
        <li>Connect to an external database to retrieve pending requests</li>
</ul>
</p>

<p>By pressing CTRL+SHIFT+T you can toggle tooltips that can help you to better understand Tucuxi.</p>
