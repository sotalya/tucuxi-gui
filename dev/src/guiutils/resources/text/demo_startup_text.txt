<p>
The current state of this demonstrator allows the user to:
<ul>
	<li>create fake patients</li>
	<li>select a medical drug</li>
	<li>select a dosage</li>
	<li>add patient's covariates</li>
	<li>add drug concentration measures</li>
	<li>propose a drug dosage adjustment</li>
</ul>

With respect to the commercial version it does not:
<ul>
	<li>save any data when the user closes the application</li>
	<li>allow to generate and edit reports</li>
</ul>
